Hórus++ é um sistema de análise preditiva de cibersegurança baseado em fontes abertas de informações na internet (OSINT).
Mas como isso funciona?

O sistema Hórus++, efetua a análise automática em 4 etapas:

Coleta de informação em fontes abertas

A coleta de informações em fontes abertas é realizada automaticamente através dos Security Bots de coleta que utilizam as preferências do cliente para buscar informações pertinentes ao seu escopo de negócio.

Análise das informações:

Utilizando técnicas de inteligência artificial e Big data, o sistema efetua a análise automática das informações coletadas identificando Entidades nomeadas, ex: Pessoas, Locais, Organizações, Software, endereços IP, domínios e ainda eventos.

Classificação

A Classificação automática evidencia o tipo de evento da mensagem, se a mesma é um vazamento de dados, um ataque de negação de serviço. As categorias observadas pelo sistema são:

Vazamento de dados (Data leak)
Desfiguração de página (Defacement)
Ataque de negação de serviço (DDOS)
Vulnerabilidades de software
Atividade de Botnet
Divulgação de ferramenta de ataque
Warez (Comércio ilegal de software, pirataria)
Notícias relacionadas a segurança da informação (Infosec)
Correlação

Através da correlação avançada de informações, utilizando inteligência artificial e análise fonética o sistema detecta as mensagens de interesse da empresa, antecipando possíveis orquestrações de ataques, novas vulnerabilidades e menções.

O sistema detecta até a grafia errada das palavras, como por exemplo o nome da empresa pessoa ou software escrito de forma incorreta, o que é muito comum na linguagem informal da Internet.

O sistema também alerta para qualquer menção a empresa envolvendo eventos relacionados a falhas de segurança, ou menções por grupos hackers.

Toda essa informação gerada pelo sistema é entregue através de uma interface Web para o analista responsável automatizando todo processo de análise de fontes abertas para cibersegurança, poupando tempo de vários analistas os deixando livres para outras tarefas.
